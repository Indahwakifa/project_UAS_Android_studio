# TUGAS UAS

```Aplikasi Pengingat Belajar adalah aplikasi sederhana yang membantu pengguna
mengingat jadwal belajar secara rutin dan teratur. Aplikasi ini dirancang
dengan antarmuka yang minimalis dan alur penggunaan yang mudah dipahami,
sesuai dengan storyboard dan desain Figma yang telah dibuat.
```

**1. Halaman Utama (Home / Splash Screen)** 

- Halaman Utama (Home / Splash Screen)

Pada halaman awal, aplikasi menampilkan:

-Logo aplikasi Pengingat Belajar

-Nama aplikasi

T-ampilan sederhana sebagai identitas aplikasi

Fungsi Halaman:

Sebagai tampilan pembuka saat aplikasi pertama kali dijalankan Memberikan identitas visual aplikasi kepada pengguna

**2. Halaman Selamat Datang**

Halaman ini menampilkan teks â€œSelamat Datangâ€ Pesan singkat pengantar penggunaan aplikasi

Fungsi Halaman:

-Memberikan kesan awal yang ramah

-Menjadi transisi sebelum masuk ke fitur utama aplikasi

**3. Fitur Pengingat Tidur**

Pada halaman ini, pengguna dapat melihat:

-Judul â€œPengingat Aktifâ€

-Informasi jadwal belajar yang telah dibuat (hari dan jam)

-Tombol Tambah Pengingat (ikon bulat / tombol +)

Fungsi Halaman:

-Menampilkan daftar pengingat belajar yang aktif

-Menjadi pusat kontrol pengingat

-Mengarahkan pengguna untuk menambahkan pengingat baru

**4. Fitur Jadwal Tidur**

Halaman ini digunakan untuk membuat pengingat belajar baru,Komponen yang tersedia:

-Pilihan hari belajar (contoh: Senin, Rabu, Jumat)

-Pilihan jam belajar

-Tombol Simpan Pengingat

Fungsi Halaman:

-Memungkinkan pengguna mengatur jadwal belajar sesuai kebutuhan

-Menyimpan data pengingat yang akan digunakan sistem

**5. Halaman Proses Tidur / Status Tidur**

Setelah pengingat berhasil disimpan, aplikasi menampilkan pesan:

-â€œPengingat berhasil dibuatâ€

-Informasi hari dan jam belajar yang dipilih

Fungsi Halaman:

-Memberi konfirmasi bahwa pengingat telah aktif

-Memberikan rasa yakin kepada pengguna bahwa data telah tersimpan

**2. Splash Screen**
Splash Screen pada aplikasi Pengingat Belajar digunakan sebagai halaman pembuka yang sederhana namun informatif, sehingga pengguna dapat mengenali aplikasi sebelum menggunakan fitur utama yang tersedia.

**Cara Kerja Fitur:**

Aplikasi membaca pengaturan bahasa / lokasi sistem pada perangkat.

Berdasarkan lokasi tersebut, aplikasi akan:

- Menampilkan bendera negara

- Menampilkan nama bahasa

- Mengubah teks â€œWelcome / Selamat Datangâ€ sesuai bahasa pengguna
